# Roadmap

## v1.1.0 (Planned)
- [ ] `git stats` command — commit count, top contributors
- [ ] `git cleanup` — remove merged local branches
- [ ] `file tree` — pretty-print directory tree
- [ ] Color output support via ANSI escape codes

## v1.2.0 (Future)
- [ ] `project scaffold` — generate project boilerplate (Node.js, Python)
- [ ] `network ping` — batch ping multiple hosts
- [ ] JSON/YAML file pretty-printer
- [ ] Plugin system for custom commands

## v2.0.0 (Vision)
- [ ] Interactive mode (REPL)
- [ ] Config file support (~/.devtoolkitrc)
- [ ] Shell completion scripts (bash, zsh, fish)
